The Blueprint Method

To count blocks without getting confused, we look at the Roofs. If we write down how high every "stack" is, we can just add the numbers together. This is like looking at a city from a bird's-eye view.

Phase 1: The X-Ray

Think of this as "Invisible Foundations." Gravity never sleeps. If a block sits on the 3rd floor, two blocks must be hiding beneath it. Success isn't about counting visible faces—it's about deducing the hidden volume required for the structure to stand.

L-Stack "Count a 2-level stack. Don't miss the corner support!"
3x3 Grid "A column is 3 high. How many 'ghost' blocks are supporting the top?"
Skyline "Calculate volume of an irregular building with hidden columns."

Phase 2: The Anatomy

The Volumetric Scanner. Toggle X-Ray vision to reveal the red **Support Blocks**. These are the foundations hidden behind neighbors. Note how the total count always includes these hidden units.

Phase 3: The Engine

Build a structure. Guess the count, then scan for hidden supports. The engine performs a "Column Audit," summing the heights of every stack roof-to-floor.

Phase 4: The Saboteur

The "Face-Value" Trap: Children often count visible "diamond" faces. A single corner block shows 3 faces (Top, Left, Right). Counting all three causes a massive overcount!

The Fix: The "Roof Tally" Rule. "Only count the roofs." Pick one gold top diamond. Ask: "What floor is this on?" Write that floor number on the diamond. **Summing the floor numbers of the roofs prevents double-counting.**
