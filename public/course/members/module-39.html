If the connection between the first pair is "A is a part of B," then the second pair must follow that exact rule without exception. If the bridge doesn't hold both pairs, the logic is broken.

Phase 4: The Saboteur

The "Inversion" Trap: This is the most common mistake. Your child might correctly identify that a finger is part of a hand, but then they pick "Toe : Foot" in the wrong order. If the first pair is Large : Small, the second pair must be Large : Small.

The Insider Fix: The "Direction Test." Tell your child: "The bridge is a one-way street." They should physically draw an arrow between the words. If the arrow goes from left to right for the first pair, it must go left to right for the second. If they say "A Finger is in a Hand," but then "A Foot is in a Toe," the logic has crashed. Order is everything.
