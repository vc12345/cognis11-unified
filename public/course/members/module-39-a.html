Module 39: Relationship Analogies

Foundation Intro

The Logic Seesaw

An analogy is just a balancing act. If the left side of the seesaw has a "Big Thing" and a "Small Thing," the right side only balances if it also has a "Big Thing" and a "Small Thing."

"Look at the first pair: **Elephant : Mouse**. That's a 'Big to Small' relationship. Click 'Test Pair' to try **Whale : Ant**. See how the seesaw stays level? The logic matches perfectly. If we tried 'Ant : Whale', the seesaw would tip because the order is wrong!"
Match the "weight" of the relationship.
Foundation Progressing

The Bridge Sentence

To find the missing word, you must build a "Bridge Sentence" for the first pair. If your sentence works for the first two words, it must work for the second two words as well.

"Look at **FINGER : HAND**. Our bridge is: 'A finger is a part of a hand.' Now use that exact sentence for **TOE**. 'A toe is a part of a... [FOOT]'. Click 'Build Bridge' to see the connection glow!"
What is the secret rule connecting the words?

Phase 1: The X-Ray

Think of an analogy as a "Logic Ratio". You aren't just matching words; you are matching relationships. The secret weapon is the Bridge Sentence. If the rule is "A is a part of B," then the second pair must obey that exact template. If the bridge doesn't hold both pairs, the logic is broken.

Phase 3: The Engine

Choose a logic type. Read the bridge out loud for both sets. If it sounds natural for both, the analogy is structurally sound.

Phase 4: The Saboteur

The "Inversion" Trap: Most children pick the right words but the wrong order. If pair 1 is Large : Small, pair 2 must be Large : Small.

The Fix: The "Direction Test." "The bridge is a one-way street." Draw an arrow. If you say "Hand has a Finger" (Left to Right), you cannot say "Toe has a Foot" (Right to Left). **Order is everything.**
