The Skeleton Matcher

To solve 11+ symbols, don't look at the shapes—look at the Repeats. If the word is "APPLE", letters 3 and 4 must be the same. Cracking the code is just finding the word that has the same repeating "skeleton" as the symbols.

"Look at this code skeleton. Positions 2 and 3 are identical. Which word matches this pattern: **CAT**, **BEE**, or **DOG**? Click the correct word to see it slide into the skeleton."

Phase 1: The X-Ray

Think of symbol mapping as "Pattern Fingerprinting." Success isn't knowing what the symbols mean—it's knowing where they repeat. If a symbol appears twice, its letter must also repeat in those exact spots. Cracking the code is a game of matching repeating skeletons.

Simple Swap "If Δ=G and Ω=O, what is the word for Δ-Ω-Δ?"
The Double Trap "Word is STEER, STARE, or STAIR. Code is Square-Star-Circle-Circle-Triangle. Which word?"
Phrase Cracker "If 'THE END' and 'THE DOG' are coded, which common symbol stands for 'T'?"

Phase 2: The Anatomy

This is the Pattern Matcher. Assign letters to the Key Bank. Watch how updating one symbol instantly updates every matching slot in the phrase. This "global update" is how a detective's brain should process logic.

Phase 3: The Engine

Experiment with the mapping. If a word stops making sense, your initial pattern match was likely wrong. Notice how every instance of a symbol changes at once.

Phase 4: The Saboteur

The "Visual Guess" Trap: Children often assume a Circle *must* be 'O' or a Bolt *must* be 'S'. Examiners use this to lead them to the wrong answer!

The Fix: The "Position Check." "The shape is a lie; the position is the truth." If a symbol is at pos 2 and 5, only look for words with matching letters at pos 2 and 5 (like LEVEL).
