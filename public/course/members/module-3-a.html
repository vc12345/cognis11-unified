Module 3: Shared Ratios (The Gap)

Foundation Intro

The "Secret Stash"

If the ratio is 3:1, it means one person has a "Secret Stash" of 2 extra boxes. When we know the "Difference" in money, that money only lives inside those Extra Boxes. It doesn't belong to the whole group!

"We both have 1 box. But you have 2 extra boxes. If those extra boxes hold £10, then every box must hold £5. See how the 'extra' money explains the 'extra' boxes?"

The Height Contest

Imagine two towers. If Tower A is 15cm taller than Tower B, that 15cm only accounts for the overhang. To find the "Magic Number" for one block, we divide that 15cm by the number of blocks sticking out at the top.

"Look at the red bracket. Tower A is taller by 3 blocks. If that gap represents 30g, how much does 1 block weigh? We only divide the gap by the extra blocks, not the whole tower!"

Phase 1: The X-Ray

In the 11+, the examiner often hides the "Total" and only tells you the Gap. Success depends on realizing that this value only lives in the Extra Boxes. If your child can see that the "Extra £20" belongs to the "2 Extra Boxes," they find the Golden Unit instantly.

The Pocket Money "Money shared 3:1. Person A has £10 more than B. How much does B have?"
The Classroom "Boys to girls ratio is 5:2. There are 15 more boys than girls. Find the girls."
The Mix "Ratio 7:3. There is 200g more Flour than Sugar. Find the total mass."

Phase 2: The Anatomy

Phase 3: The Engine

Adjust the "Gap Amount" and the "Ratio." Watch how the value of one box is calculated only by using the difference between A and B.

Phase 4: The Saboteur

The "Total Blindness" Trap: Children often see a number and immediately add the ratio parts together (e.g., 5 + 3 = 8). This is a disaster if the number is actually a Gap.

The Fix: The "Minus First" Rule. If you see "more than" or "difference," you must subtract the ratio numbers first. The "Gap Money" only fits into the "Gap Boxes."
