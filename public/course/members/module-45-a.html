Phase 1: The X-Ray

Think of a matrix as "The Logic Intersection." Every cell is governed by two separate laws: the Row Rule (horizontal change) and the Column Rule (vertical change). The missing piece is the fusion of these rules. If Row 3 is "Circles" and Column 3 is "Green," the answer is locked.

Pattern Match "Row 1: Red Sq, Blue Sq, Green Sq. Row 2: Red Tr, Blue Tr, [?]."
Rotation Grid "Elements spin 90° clockwise in every successive cell."
11+ Fusion "Size increases down columns, while color alternates across rows."

Phase 2: The Anatomy

The Attribute Intersection. Notice that shape is constant per row (Triangles, Squares, Circles). The target cell must fit Row 3's shape and the column's property. The answer is "pinned" by both axes.

Phase 4: The Saboteur

The "One-Track" Trap: Children find one pattern (e.g. "it's a circle row") and pick the first circle they see, ignoring color or tilt.

The Fix: The "Cross-Hair" Method. "Split the evidence." Isolate Attribute 1 (Shape) via rows. Isolate Attribute 2 (Color/Tilt) via columns. **The answer is the sum of both facts.**
