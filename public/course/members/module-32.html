Module 32: Symbol Mapping

Phase 1: The X-Ray

Think of symbol mapping as "Pattern Fingerprinting." In the 11+, you are given a code made of strange shapes and asked to match it to a word. The secret weapon isn't knowing what the symbols mean—it's knowing where they repeat. If a symbol appears twice, the letter it hides must also appear twice in those exact same spots. Cracking the code is simply a game of finding the word with the same repeating "skeleton" as the symbols.

The Simple Swap "If Δ = G and Ω = O, what 3-letter word is represented by Δ-Ω-Δ?"
The Double-Letter Trap "The word is STEER, STARE, or STAIR. If the code is Square-Star-Circle-Circle-Triangle, which word is it?"
The Phrase Cracker "Two codes: 'Ψ-Ω-Σ' and 'Ψ-Δ-Γ'. If the words are 'THE END' and 'THE DOG', which symbol stands for 'G'?"

Phase 4: The Saboteur

The "Visual Guess" Trap: Most children look at a symbol and try to guess the letter based on its shape (e.g., assuming a Circle must be 'O' or a Bolt must be 'S'). This is a trap set by examiners to lead them to the wrong answer.

The Insider Fix: The "Position Check." Tell your child: "The shape is a lie; the position is the truth." If a symbol appears at position 2 and 5 of a word, they are only looking for words with the same letter at position 2 and 5 (like LEVEL). Ignore the drawing; count the slots.
